ArrayList简介

对于Java开发人员来说，ArrayList是最常用的数据结构之一。它是List接口的一个具体实现类。ArrayList是容量可以改变的非线程安全集合。内部实现使用数组进行数据存储，集合扩容时会创建更大的数组空间，在将原有数据复制到新数组中。ArrayList实现了RandomAccess接口从而支持对元素的快速随机访问，但是插入与删除时速度通常较慢（因为删除和插入的过程可能需要移动元素）。ArrayList允许null元素存入，ArrayList的iterator和listIterator方法返回的迭代器是fail-fast的。
